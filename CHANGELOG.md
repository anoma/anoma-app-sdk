# Changelog

## [0.2.0](https://github.com/anoma/pay-interface-app/compare/v0.1.0...v0.2.0) (2026-04-30)

This is the initial CHANGELOG for the AnomaPay interface app. It includes all the changes made since the initial release (v0.1.0) up to the current version (v0.2.0).

The current state contains: authentication, dashboard, deposit, send, request and pay pages, transaction history, in-app notifications, and more. The app is still in public beta, and we are continuously working on improving it based on user feedback and testing.
